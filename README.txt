Sports Management System
Overview
This is a Sports Management System web application that allows users to manage sports-related activities, including registering, logging in, viewing coaches, viewing members, managing inventory, and accessing a dashboard with essential features for managing a sports club.

Features
Admin Dashboard: Manage the system with administrative control.

Registration and Login: Secure user registration and login.

Inventory Management: View and manage the available sports inventory.

View Coaches: View the list of available coaches.

View Members: View and manage registered members.
